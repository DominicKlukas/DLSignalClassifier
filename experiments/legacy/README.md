# Legacy Experiments

These directories preserve older branches, baselines, and side paths that are not part of the cleaned main experiment sequence, but are still worth keeping for provenance and comparison.

The naming is by historical role rather than by the original flat filename:

- `exp02_plain_multimodal`: early multimodal fusion before the gated version
- `exp04_spectrogram`: spectrogram-only branch
- `exp05_orbit_rf`: ORBIT benchmark branch
- `exp05_subghz_real`: early raw Sub-GHz real-data branch
- `exp06_real_h5`: HDF5-based comparable benchmark runs and related result files
- `exp07_awgn_ablation_real`: real-data AWGN ablation
- `exp08_awgn_ablation_waveform`: waveform-family AWGN ablation
- `exp09_wavelet_multidataset`: wavelet and multidataset branch

Compatibility wrappers remain at the old top-level script names.
